# 🤖 AI ASSISTANT

A futuristic Python-based Voice Assistant built using Python 3.14 and PyQt6.

## ✨ Features

- 🎤 Voice Recognition
- ✨ Floating Particle Effects
- 🕒 Tell Time and Date
- 🔋 Battery Percentage
- 🌐 Open Google, YouTube

## 🛠 Technologies Used

- Python 3.14
- PyQt6
- SpeechRecognition
- pyttsx3
- psutil
- requests
- Pillow
- PyAutoGUI
- NumPy
- SoundDevice

## 📦 Installation


pip install PyQt6 SpeechRecognition pyttsx3 psutil requests pillow pyautogui numpy sounddevice pyaudiowpatch


## ▶️ Run Project


python main.py


## 🎙 Example Commands

- What time is it
- Battery percentage
- Open Google
- Open YouTube
