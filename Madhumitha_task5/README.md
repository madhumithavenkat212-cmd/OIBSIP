# 🤖 Cyber AI ChatBot

A futuristic browser-based AI chatbot built using **Python Flask**, **HTML**, **CSS**, and **JavaScript**. The application features a modern cyberpunk-inspired interface with Matrix-style animations, real-time chat functionality, and a responsive design.

---

## 📌 Features

- 💬 Browser-based AI Chat Interface
- 🐍 Python Flask Backend
- ⚡ Real-time Chat Communication
- 🌌 Matrix Animated Background
- 🔵 Neon Cyberpunk UI Design
- ⌚ Live Digital Clock
- ✨ Typing Animation Effects
- 📱 Fully Responsive Design
- 🔔 Interactive Notifications

---

## 🛠️ Technologies Used

- **Python**
- **Flask**
- **HTML5**
- **CSS3**
- **JavaScript**
- **Google Fonts**
