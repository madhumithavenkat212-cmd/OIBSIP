# 🌦️ Weather Application

A simple and responsive Weather Application built using HTML, CSS, and JavaScript. This application allows users to search for weather details of any city.

## RUN
by index.html

## ✨ Features
- Search weather by city name
- Displays temperature
- Shows humidity and wind speed
- Displays weather conditions with icons
- Responsive design

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
- Weather API
