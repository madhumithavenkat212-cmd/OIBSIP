# 🩺 AI Smart BMI Health System

An AI-powered BMI (Body Mass Index) Health Monitoring Web Application built using **Python Flask**.

## 🚀 Features

- ✅ BMI Calculation
- ✅ Health Category Detection
  - Underweight
  - Normal
  - Overweight
  - Obese
- ✅ Personalized Health Tips
- ✅ Daily Water Intake Recommendation
- ✅ Daily Calorie Recommendation
- ✅ BMI History Storage using SQLite Database
- ✅ BMI Trend Chart Visualization
- ✅ Hacker-Style Animated UI
- ✅ Glassmorphism Design
- ✅ Responsive Web Interface

## 📊 How It Works

1. Enter:
   - Name
   - Age
   - Gender
   - Height
   - Weight

2. Click **Calculate BMI**.

3. The system:
   - Calculates BMI
   - Determines health category
   - Gives health recommendations
   - Stores history in database
   - Displays statistics and charts

---

## 🧮 BMI Formula


BMI = Weight (kg) / Height² (m²)
