🌟 Oasis Infobyte Internship Projects

This repository contains the projects completed as part of the Oasis Infobyte Internship Program. These projects were developed to enhance practical knowledge in web development, Python programming, and user interface design.

---

📌 Projects Included

1️⃣ Weather Application

A responsive weather application developed using HTML, CSS, and JavaScript that provides real-time weather information.

Features

- Search weather by city name
- Displays temperature, humidity, and wind speed
- Dynamic weather icons
- Responsive user interface

Technologies Used

- HTML5
- CSS3
- JavaScript
- Weather API

---

2️⃣ AI Voice Assistant

An intelligent desktop-based voice assistant developed using Python.

Features

- Voice recognition
- Opens websites and applications
- Provides current time and date
- Executes basic system commands

Technologies Used

- Python
- pyttsx3
- SpeechRecognition
- Tkinter

---

3️⃣ BMI Calculator
